Quick Start


1
What are the components of Megam ?
There are three major components Medium,Cloud Gateway, Cloud Herk
Medium:The Medium can be WebApp built using Rails, CLI (Ruby) or Mob (Sencha Touch * coming soon ).
Cloud Gateway : Cloud Gateway is a RESTful Play(Scala) responsible for managing the description of cloud resources.
Cloud Herk: Cloud Herk is an instrumentor by taking the cloud description and mediates between the PubSubEngine and the Cloud Gateway
