Features

Whether you need to create a PaaS from scratch, including setting up a small office or rehosting your application lifecycle (SCM, cloudifying your traditional infra).

Visible PaaS

Embraces developers and DevOps and not be a black box that strips away IT control and visibility.

Pluggable Architecture

Pluggable architecture from the inside out so that the system grows and expands with increased and diverse deployments. DevOps can extend the deployable maker based on the their IT Policy.

Infra Agnostic

It is infrastructure agnostic leveraging existing IT infrastructure and investments to avoid vendor lock-in (Support for openstack, rackspace, AWS for the beta launch)

Features 2

Whether you need to create a brand from scratch, including marketing materials and a beautiful and functional website or whether you are looking for a design refresh pleased with the results.

Seamless interaction

Seamless interaction model between users of the system. No system forced policy in the client libraries for developers.

Instant Onboarding

Frictionless onboarding that does not require a rewrite to your code base or the addition of large metadata that is specific to the system.

SCM Connectors (GITHub)

Injectable dependencies for the app from external means. (SCM - source control management system supported to start with is Github. More connectors to be added.)

Multi build engine

Multi build engine support (eg: Java - MVN (Maven), Play(Scala) - MVN, SBT(Simple build tool), Node.js (npm), RoR (Ruby on rails) - Bundle).

Quick App Provisioning

Quick app provisioning for IT using an UI (userinterface), CLI (Command line interface).

Roadmap(Mobile & SCM Support)

Mobile single click deployment and monitoring and support for other SCMS are in the road map.
